Admin för Berga Fönsterputs

Hantera bokningar, Swish och RUT.

Här kan ni se bokningar, skicka Swishuppgifter i klartmejlet, kontrollera betalningar och hämta RUT-underlaget när det är dags att ansöka.

Det här kan ni göra här

  • Se alla bokningar på ett ställe
  • Skapa, pausa och följa upp rabattkoder
  • Lägga in semesterdagar som ej bokbara
  • Markera bokning som bekräftad
  • Markera jobb som klart och skicka Swishuppgifter
  • Markera bokning som betald efter kontroll i Swish Företag
  • Visa och gallra krypterade RUT-uppgifter

Logga in

Använd ett Supabase-konto för admin. Efter inloggning kan ni hantera bokningar direkt här.

Tvåstegsverifiering

Admin innehåller känsliga kunduppgifter. Bekräfta därför inloggningen med en kod från din authenticator-app.

Skanna koden i din authenticator-app Om du inte kan skanna, skriv in nyckeln manuellt: Registrera gärna samma nyckel i en andra betrodd app eller enhet som reserv innan du går vidare.

Semesterdagar

Lägg in ett datum eller ett intervall som inte ska kunna bokas. De spärras både i kalendern och när bokningen skickas.

Rabattkoder

Skapa en kod som kunden kan ange i bokningen. Rabatten kan vara procent eller ett fast belopp i kronor.

Bokningar

Här visas alla bokningar, även slutförda och betalda, så att inget försvinner ur adminvyn.
